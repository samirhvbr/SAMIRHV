GitHub Desktop

Linux Windows v3.5.15 Beta 3 atualizado 02 jun 2026 x64 · arm64

GitHub Desktop é o cliente Git visual e open-source da GitHub — construído em Electron e escrito em TypeScript com React. Commits, branches, histórico, pull requests e resolução de conflitos numa interface limpa, sem decorar comandos.

Oficialmente a GitHub não distribui o app para Linux. Estes são builds da comunidade que compilam o GitHub Desktop a partir do código-fonte e empacotam para cada sistema — prontos pra instalar.
